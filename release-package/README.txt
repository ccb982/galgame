Galgame 播放器 - 便携版
========================

系统要求
--------
- Windows 7 或更高版本
- Java 17 或更高版本（如果未安装，启动时会提示下载）

快速开始
--------
1. 双击"启动.bat"文件
2. 等待服务启动（约 5 秒）
3. 浏览器会自动打开 http://localhost:8080

功能说明
--------
- 游戏库：浏览和管理游戏
- 游戏详情：查看游戏信息和存档
- 游戏播放器：体验视觉小说
- 管理后台：上传游戏、剧本和图片

目录结构
--------
galgame-backend-1.0.0.jar  - 后端服务程序
frontend/                       - 前端静态文件
启动.bat                       - 启动脚本
data/                          - 数据库文件（首次运行自动创建）
uploads/                       - 上传的图片文件（首次上传自动创建）

注意事项
--------
1. 首次运行会自动创建数据库文件
2. 数据库文件保存在 data 目录
3. 上传的图片保存在 uploads 目录
4. 关闭命令行窗口即可停止服务
5. 端口 8080 被占用时无法启动

已知问题
--------
- 启动时可能会看到一些关于 user 表的 SQL 警告，这是正常的，不影响使用
- user 表是预留的用户认证功能，当前版本未启用

常见问题
--------
Q: 提示未找到 Java 环境？
A: 请安装 Java 17 或更高版本，下载地址：https://www.oracle.com/java/technologies/downloads/

Q: 端口 8080 被占用？
A: 修改 galgame-backend-1.0.0.jar 内的配置文件，或关闭占用 8080 端口的程序

Q: 如何查看数据库？
A: 访问 http://localhost:8080/h2-console，使用默认连接信息登录

Q: 如何备份数据？
A: 备份 data 目录和 uploads 目录即可
